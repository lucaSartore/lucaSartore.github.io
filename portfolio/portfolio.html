Puzzle Solver

Check Out

I developed an algorithm that can solve small-sized puzzles. The algorithm takes as input one or more images containing pieces of a shuffled puzzle, captured using a printer's scanner. It utilizes computer vision and an algorithm to find the solution to the puzzle.

The app is built using three different languages:

- C# was used to develop the UI.
- C++ was employed for the computer vision component, leveraging the OpenCV library.
- Rust was utilized for the solution finder itself.

The entire program is designed to scale efficiently with multiple cores and is dynamically linked using DLLs."

My RoboCup Robot

Check Out

A robot developed by me and four other people, for the RobCup junior competition.

The robot is entirely designed in 3D and is made of 30+ pieces, and is 3D printed using a combination of PLA and resin.

In total the robot has 11 motors, 4 ultrasonic distance sensors and one camera.

The program runs on a raspberry PI and is made in python using the OpenCV library We have also trained a custom machine learning model for object detection, and used a Google Coral TPU as an hardware accelerator, since the raspberry pi was too slow.
